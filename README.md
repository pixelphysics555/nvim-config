# slick-nvim

## Goal: Create a one-step-install neovim configuration that contains a handful of useful plugins that can be used in any UNIX operating system.

## Plugins:
* Alpha Dashboard
* Code Completion
    - Nvim-cmp, Cmp-path, Cmp-nvim-lsp
    - Luasnip, Cmp-luasnip
* Conform
* LuaSnip
* LSP
    - Mason
    - Mason-lsp
    - Fidget
    - Neodev
* Lualine
* Neo-tree
* Telescope
* Colorscheme
    - Tokyonight
    - Gruvbox
    - Dracula
* Treesitter
* VimTeX
